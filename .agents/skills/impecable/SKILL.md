# Impecable Skill

Diretrizes para manter o desenvolvimento de código e interfaces no estado da arte:

## 1. Qualidade de Código (C#)
- **Legibilidade**: Nomes de variáveis descritivos, métodos pequenos com uma única responsabilidade.
- **Tratamento de Erros**: Sempre proteger operações de arquivo, rede e processos externos com blocos `try-catch` robustos e amigáveis.
- **Escopo Limpo**: Respeitar o escopo de variáveis locais em relação a construtores primários de C# 12+.

## 2. Design e Interfaces Web (HTML/CSS)
- **Estética Premium**: Uso consistente de gradients, shadows (como o glow de tecnologia), micro-animações nas interações e paletas escuras elegantes.
- **Responsividade**: Garantir alinhamento perfeito em resoluções desktop e mobile.
- **Fluidez**: Carga rápida com cache instantâneo e varredura incremental em background sempre que aplicável.

## 3. Scripts de Sistema e CLI
- **Compatibilidade de Encoding**: Garantir suporte a caracteres especiais no Windows CMD usando `chcp 65001 > nul` no topo dos arquivos batch (`.bat`/`.cmd`).
- **Feedback Visual**: Divisores bem formatados usando traços simples (`--------------------`) e mensagens de progresso claras em cores contrastantes.
